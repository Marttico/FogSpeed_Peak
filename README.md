# FogSpeed

This mod does what it says on the tin! You can speed up or slow down the fog speed.

[Thunderstore page](https://thunderstore.io/c/peak/p/TeamCo/FogSpeed/)

Default keybinds:

- `J` Slow down Fog
- `K` Pause fog
- `L` Speed up Fog
- `I` Advance fog
- `O` Change fog direction

Config parameters:

- Default fog speed (By default `0.3`)
